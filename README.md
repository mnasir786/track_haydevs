# TrackFlow Desktop Agent

Download the latest version from the [Releases page](https://github.com/mnasir786/track_haydevs/releases/latest).

## Which file to download?

| Platform | File |
|----------|------|
| macOS Apple Silicon (M1/M2/M3) | `TrackFlow-v1.0.56-arm64.dmg` |
| macOS Intel | `TrackFlow-v1.0.56.dmg` |
| Windows | `TrackFlow.Setup.1.0.56.exe` |

Not sure which Mac you have? Click **Apple menu > About This Mac** and look for "Apple M" (Silicon) or "Intel" in the chip line.

## macOS Installation

1. Download the correct `.dmg` for your chip (arm64 = Apple Silicon, no suffix = Intel)
2. Open the `.dmg` file
3. Drag **TrackFlow** to **Applications**
4. Launch TrackFlow from Applications
5. Sign in with the credentials sent by your administrator

> If macOS says the app cannot be opened: go to **System Settings > Privacy and Security** and click **Open Anyway**

## Windows Installation

1. Download `TrackFlow.Setup.1.0.56.exe`
2. Run the installer
3. If Windows SmartScreen appears, click **More info > Run anyway** — this is expected for new apps
4. Sign in with the credentials sent by your administrator

## About

TrackFlow is a desktop time tracking agent. This repository contains official release binaries only.
Source code is maintained privately.
